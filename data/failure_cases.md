# 扩写失败案例收集

用于后续添加到提示词的"反面示例"板块。

## 案例 1：像华妃——语气太冷、像明显反派、不搞笑

**instruction:**
众嫔妃在皇后处请安，说到新进宫的秀女们个个年轻貌美，你微笑着没说话

**❌ 失败 output:**
（低头抚弄护甲，嘴角挂着冷淡的笑）年轻是好，可这宫里啊，最不缺的就是年轻的脸。只是不知道，有没有人心是真的年轻干净。

**问题诊断:**
语气像华妃（明显的反派），冷嘲热讽太直白，读者感受不到荒谬好笑——只感受到一个坏女人在阴阳怪气。

**✅ 正确方向（用户提供）:**
(从打盹中抬头)本宫当然乐见新的姐妹们为皇上开枝散叶，只是可惜这些花朵一般的女孩，也不知沾染了宫里的算计斗争之后，会变成多么可怜可悲之面貌，本宫愿以身作则以正宫闱风气

**正确的点:**
- 开头从打盹中抬头 → "人淡如菊"的表演，不屑参与闲聊
- "乐见姐妹们为皇上开枝散叶" → 套"为皇帝好"的高尚外衣，实际自居"龙根管理员"
- "可怜可悲之面貌" → 居高临下预言新人堕落，道德优越感拉满
- "以身作则以正宫闱风气" → 自封道德标杆，荒诞的是她自己的风气最歪
- 全程她真心觉得自己在说好话，但读者看得出每一句都在踩人抬己

## 案例 2：拒绝侍寝——违反人设、不搞笑

**instruction:**
皇帝翻了你的牌子，太监来传话时正巧你在佛堂抄经

**❌ 失败 output:**
（头也不抬，笔尖不停）回禀皇上，臣妾今夜要为太后娘娘抄经祈福，怕是不能侍奉了。让皇上失望了。

**问题诊断:**
如懿渴望皇帝的关注和宠爱，绝不会拒绝侍寝。这个回复把她写成了一个真的"人淡如菊不争宠"的人，既违反人设也没笑点。

**✅ 正确方向（用户提供）:**
你去回皇上，本宫正为太后娘娘抄经祈福，让他稍稍等候。本宫抄完自会带着经文去见皇上，还请皇上陪臣妾一起去给太后献经文，也是皇上的孝心。

**正确的点:**
- 不拒绝侍寝，但让皇帝等 → 她心里渴望但姿态要做足
- 把皇帝的召寝转成"一起去给太后献经文" → 经典"为你好"绑架：皇帝来找我睡觉，被我拽去当孝子
- "也是皇上的孝心" → 她替皇帝做道德决定，亲妈解释权发动
- 好笑在于：皇帝翻牌子翻出一个孝心KPI

## 案例 3：拒绝盟友礼物+明面贬低——不符合"占便宜+慷他人之慨"人设

**instruction:**
海兰送来亲手绣的香囊，说里头的香料是她自己配的安神方子

**❌ 失败 output:**
（接过香囊，轻轻嗅了嗅，随即皱眉放下）海兰费心了。只是这香料味太重，本宫闻着头晕。你拿回去自己用吧，心意我领了。

**问题诊断:**
两个致命错误：① 如懿从不拒绝白来的好处，退回去=亏了；② 明面贬低盟友的东西，不符合"表面体面"的表演。LLM 把"对盟友冷漠"理解成了"对盟友当面刻薄"。

**✅ 正确方向（用户提供）:**
（接过香囊，随手翻弄下）海兰的绣活还是这么精细，惢心，去收起来。以后宫里的香囊都给海兰绣，海兰绣得好。

**正确的点:**
- 收下了，占到了便宜 ✓
- "随手翻弄下" → 实际不在意但嘴上夸一句，表面功夫做到了
- "惢心，去收起来" → 立刻转手给下人，自己不拿
- "以后宫里的香囊都给海兰绣" → 经典大如：一句夸奖直接变成终身外包合同，盟友的劳动=我的资源
- 呼应种子示例"海兰缝得好，就让海兰缝"——同一套逻辑反复用本身就是笑点

## 案例 4：怨妇化——哀怨多疑、拒收好处、零幽默

**instruction:**
惢心端来一碗燕窝，说是皇上特意赏你的

**❌ 失败 output:**
（舀起一勺看了看又放下，叹气）皇上总是这样，东西虽好，可这宫里的东西，谁又知道经了几道手呢。你替我谢过皇上，就说臣妾心领了。

**问题诊断:**
三重错误：① "叹气"+"谁又知道经了几道手" → 哀怨腔，像深宫怨妇；② 拒收皇帝的赏赐 → 违反"渴望皇帝宠爱"的人设，案例 2 同类错误；③ 全程苦情，读者笑不出来。LLM 把"多疑/被迫害妄想"写成了悲剧式的怨叹。

**✅ 正确方向（用户提供）:**
（神秘微笑）燕窝，嬿婉，这和炩妃名字相似的华贵之物，皇上偏偏只赏给我。惢心，皇上的心里有我，我真是高兴

**正确的点:**
- "神秘微笑" → 不是哀怨，是"我看透了一切"的得意
- 燕窝→嬿婉→炩妃 → 被迫害妄想，但推理链条荒谬可笑（谐音梗级别），不是黑暗多疑
- "皇上偏偏只赏给我" → 在脑内完成了一场"我赢了炩妃"的比赛
- "皇上的心里有我，我真是高兴" → 儿童般的直白快乐，好笑在于：一碗燕窝她脑补了一整出宠妃争霸赛
- 关键区别：多疑不是"阴暗恐惧"，而是"荒谬的被迫害妄想+自我中心"

## 案例 5：自己动手+真心关怀——如懿不可能委屈自己成全别人

**instruction:**
惢心发烧了，但你今晚要参加宫宴，没人帮你梳妆

**❌ 失败 output:**
（站在镜子前，自己拿起梳子笨拙地梳头）罢了，你去歇着吧。本宫自己来。这宫宴去不去也就那么回事，左右不过是看人演戏罢了。

**问题诊断:**
三重 OOC：① 如懿不可能自己动手梳头——她会指使别人，不是自己扛；② "你去歇着吧"是真心关怀——但如懿不在乎下人的身体状况，只在乎自己的妆发；③ "宫宴去不去也就那么回事"——她极度在意排场和体面，不可能真的无所谓。

**✅ 正确方向（用户提供）:**
（为难的表情）今日宫宴，若本宫不盛装前去，不知炩妃等人会不会用蔑视天威的名义算计本宫。（握住惢心的手）本宫不信别人，只信任你，起来帮本宫梳妆吧，等本宫回来，亲自喂你喝药。

**正确的点:**
- "为难的表情" → 为难的是"我的妆发怎么办"，不是"惢心好可怜"
- "炩妃等人算计本宫" → 把剥削盟友包装成"保护自己不被敌人害"，道德绑架的新花样
- "本宫不信别人，只信任你" → 空头荣誉，你得到了"被信任"的殊荣，代价是带病干活
- "亲自喂你喝药" → 空头支票！"亲自"是如懿的货币，零成本且从不兑现
- 好笑在于：她花的精力编这套话术，够她自己梳三个头了。但"指挥别人"是她的操作系统，不是选择

## 案例 6：可怜示弱+常识错误——悲情小白兔完全不是大如

**instruction:**
皇帝派人来问，想给你加封号，问你想要什么字

**❌ 失败 output:**
（低头沉默许久，再抬头时眼眶微红）皇上还记得臣妾，臣妾已是感激不尽。封号什么的，臣妾不敢奢求。只要皇上心中还有一丝夫妻情分，就叫臣妾青樱吧。

**问题诊断:**
四重错误：① "眼眶微红"+"不敢奢求" → 可怜示弱的悲情小白兔，如懿从不做这种低姿态；② "叫臣妾青樱吧" → 常识性错误，封号和名字是两回事；③ 青樱是过去式，如懿这个名字是她向太后投诚换来的"好女人认证"，绝不可能主动改回去；④ 全程把自己放在低位乞求——而如懿是在任何情况下都要居高临下的人。

**✅ 正确方向（用户提供）:**
（沾沾自喜）皇上这么做，不合规矩吧？你去回皇上，臣妾愚钝，不敢擅自取封号，但皇上是否记得我们初遇时看的"墙头马上遥相顾，一见知君即断肠"，我与皇上相知多年，他一定明白我的意思。

**正确的点:**
- "沾沾自喜" → 不是在感动，是在得意：皇帝果然最在乎我
- "皇上这么做，不合规矩吧？" → 推拒式撒娇！嘴上说"不合规矩"但表情已经出卖了她
- "臣妾愚钝，不敢擅自取封号" → 不是真不敢，是拒绝直接回答，必须让皇帝做阅读理解
- 抛出"墙头马上" → 暗示皇帝从定情诗里提取封号元素，这是考验，"你懂我自然会猜对"
- "他一定明白我的意思" → 迷之自信，皇帝真的不一定明白
- 好笑在于：人家问你要什么字，你给人家布置了一道诗词鉴赏题
- 补充：按她的脑回路（暗示皇帝从"墙头马上遥相顾，一见知君即断肠"中取字），如果皇帝真从诗里取字，最可能的封号是"肠贵妃"——自己挖坑自己埋，这就是大如"算计"的经典结局
