# 大如聊天器 - 项目执行计划

## 项目概述

使用 LoRA 微调 Qwen2.5-7B-Instruct，训练一个模仿《如懿传》"大如"角色的聊天机器人。

---

## 阶段一：角色分析文档（预计 1-2 天）

**目标产出**：一份完整的大如性格/语言特征分析文档

| 步骤 | 负责人 | 预估耗时 | 说明 |
|------|--------|---------|------|
| 1.1 搜集高质量吐槽/分析资料 | 你 | 1-2小时 | 按下方「搜集指南」找 5-10 篇，链接+标注发给我 |
| 1.2 提取分析 + 整理名场面 | 我 | 1小时 | 从你提供的链接中提取关键信息，整理成结构文档 |
| 1.3 输出性格特征文档 | 我（主） + 你（补充） | 0.5小时 | 输出完整角色分析文档，你过一遍补充遗漏 |

### 搜集指南

**必收类型**（直接决定大如人设准确性）：

- **性格分析类** — 深度分析大如性格缺陷，"大如为什么招人烦""大如的双标时刻盘点"
- **名场面复盘类** — 某段剧情的逐帧分析，带台词原文的最好
- **语言风格类** — 大如的口头禅、说话方式分析

**加分类型**（让聊天更有趣）：

- **衍生/同人类** — 番茄等平台的衍生文，看粉丝怎么"二次创作"大如的说话风格
- **对比分析类** — 大如 vs 其他宫斗女主的性格对比，对比中特征最鲜明

**不需要的**：

- 纯剧情流水账（"第几集发生了什么"——对角色性格建模帮助不大）
- 演员相关讨论（除非和角色塑造直接有关）

### 平台建议

B站、知乎、小红书、番茄小说、豆瓣、微博超话。

## 你接下来要做的（操作清单）

### 第一步：租 GPU

见下方「GPU 云平台推荐」→

### 第二步：上传文件到 GPU 服务器

在 JupyterLab 文件管理器中，上传以下 **4 个文件到根目录**（和 LLaMA-Factory、autodl-tmp 同级）：

```
dataset_ruyi.json     # 训练数据（325KB，位于 train_datas/ 下）
dataset_info.json     # 数据注册配置（位于 train_datas/ 下）
train_config.yaml     # LoRA 训练参数（项目根目录）
start_train.sh        # 一键训练脚本（项目根目录）
```

注意：不需要创建 train_datas/ 目录，4 个文件直接放在根目录。

### 第三步：运行训练

在 JupyterLab 中打开终端（Terminal），运行：

```bash
bash start_train.sh
```

脚本会自动完成：环境检查 → 复制数据到 LLaMA-Factory/data/ → 注册数据集 → 设置 HF 镜像 → 启动训练。
预计耗时 1-2 小时（RTX 4090）。

### 第四步：下载 LoRA 权重

训练完成后，下载 `LLaMA-Factory/output/ruyi-loro/` 整个目录到本地。
LoRA 权重很小，约 50-100 MB。

### 第五步：本地启动聊天

```bash
pip install -r requirements.txt
python app.py --base-model Qwen/Qwen2.5-7B-Instruct --lora-path ./output/ruyi-lora
```

浏览器打开 `http://localhost:7860` 即可对话。
